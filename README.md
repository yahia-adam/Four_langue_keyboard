# pooring_keyboard

Ce projet est une solution de saisie numérique dédiée à la langue Four (Sárí), originaire du Darfour au Soudan. L'objectif est de briser les barrières technologiques en offrant un clavier virtuel intuitif qui intègre nativement les caractères spéciaux, les tons et les spécificités linguistiques indispensables à cette langue. En facilitant l'écriture du Four sur le web et les mobiles, cet outil contribue directement à la préservation de l'identité culturelle et au développement de la communication numérique pour toute la communauté.
